如何修改微信步数？

发表于 2018-12-06 | 更新于: 2018-11-27 | | | 次阅读

准备工作：Mac电脑、Xcode

步骤：

1、前往Github下载class-dump

https://github.com/nygard/class-dump

2、编译（command + B），Show in Finder，将文件拷贝出来，前往/usr/local/bin/class-dump，将文件放入该目录；

3、打开终端，输入class-dump
